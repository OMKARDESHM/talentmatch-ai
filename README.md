
# TalentMatch AI

TalentMatch AI is an AI-powered job board that helps candidates discover relevant job opportunities using natural-language search and explainable job matching.

## Project Status

The project is under active development as part of a time-boxed engineering assignment.

## Planned Core Features

- Company Admin job management
- Candidate profile management
- Job search and filtering
- AI-powered natural-language job matching
- Explainable job ranking
- Candidate application workflow
- Application pipeline management
- Admin recruitment analytics
